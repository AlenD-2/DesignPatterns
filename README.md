# C++ Design Patterns Example

An implementation of some common Design Patterns based on "Design Patterns in Modern C++" book.

## Content

These Examples include:

- Builder
- Groovy
- Prototype Factory
- Composite Builder
- Abstract Factory
- Factory
- Copy Construction Prototype
- Classic Singelton
- Adapter
- Array Backed Properties
- Group Object
- Dynamic Decorator
- Static Decorator
